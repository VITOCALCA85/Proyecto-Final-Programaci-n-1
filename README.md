# Proyecto-Final-Programación 1

**Inventario de Mapas para Primaria y EBI:  es una aplicación en Python con interfaz gráfica (Tkinter) y base de datos SqLite.

En este proyecto final, pensamos en un sistema ágil para para consultar, registrar, categorizar, almacenar y buscar mapas. 

En el inventario aparecerá la siguiente información sobre cada mapa:título,fecha,autor,tipo y el enlace al archivo de cada mapa. Pensando que cada mapa pueda descargarse.

Podría ser usado por Maestras en las escuelas y docentes de Historia/Geografía en EBI.

*Utilizamos: -tkinter (interfaz gráfica)
             -sqlite (persistencia)
             -pytest (para pruebas unitarias, opcional)

La técnica del encapsulamiento para proteger los datos y acceder a través de métodos públicos. Este inventario permite la búsqueda y organización adecuada en Primaria y Secundaria. Se utilizan atributos privados y se controlan con con "getters" y "setters". Esto es clave para enseñar buenas prácticas de POO.
Los docstrings explican qué hace cada clase.
El mapa representa un recurso educativo.
El "InventarioMapas" gestiona un conjunto de recursos.
Hay principios de diseño limpio. Al buscar por título se usar "lower" para búsquedas no sensibles a mayúsculas y minúsculas.

En la actualidad el mapa es un recurso muy utilizado por nosotros ya sea en el aula o en tareas domiciliarias ya que permite que el estudiante pueda itnerpretar los espacios geográficos o procesos históricos de manera vidual y significativa.
Fomenta el uso de recursos digitales para explorar el territorio de manera interactiva permitiendo al estudiante apropiarse del recurso y conceptos.
Si este tipo de inventario se asociara al uso de la itneligencia artificial seria de mucha utilidad en las aulas para promover la búsqueda autonoma, la resolución de problemas, aplciación de cocneptos y el abordaje desde la interdisciplinariedad.

Se investigó de como insertar las imágenes de los mapas en python y  que no se enocntró la forma, por ello se decidió agregar los  link para que puedan ser visualizados desde las páginas web y ser descargados.
Las librerías que elegimos son: "tkinter" y "sklite" porque la primera nos permite crear una ventana principal que es el Inventario de mapas y después ingresar datos (nombre, año, tipo...) y nos permite usar botones: agregar mapa, buscar por título y hacer una lista. La segunda nos permite guardar información de forma permanente y segura. 
La utilización de la inteligencia artifical para el proyecto ha sido de guía.
